# Neural Style Transfer Web App
- A simple web application for neural style transfer using AdaIN algorithm.
# Requirement
- Python 3.10
# How to Use
1. Download and extract the files.
2. Run the command prompt as an administrator.
3. Install the required packages using the following command:
   
   ```Bash
   pip install -r requirements.txt

4. Go to the following Google Drive:

   ```Bash
   https://drive.google.com/drive/folders/1TyFziQZGmi86SK4fHFSWW1N_tf4iSDmk?usp=sharing

5. Download the folder `models` in the Google Drive. The model weights `vgg_normalized.pth` and `decoder.pth` are stored in the folder.
6. Put the folder `models` in the same location with the Python script `neural_style_transfer.py`.
7. Run the Python script `neural_style_transfer.py` to start the web application.
8. Open your web browser and go to `http://127.0.0.1:7860` to access the web application.
